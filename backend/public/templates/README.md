# Plantilla para Carga Masiva de Usuarios

Este directorio contiene las plantillas Excel para importación de usuarios.

## Archivos

- `plantilla_usuarios.xlsx` - Plantilla oficial para carga masiva

## Uso

1. Descargar la plantilla
2. Completar con los datos de los usuarios
3. Guardar el archivo
4. Subir en la sección Usuarios → Importar
